# Project Incubator — 新規プロジェクト企画室

新しいビジネスアイデアのリサーチ・検討・PoC（概念実証）を行う場所。

## 評価基準

新規プロジェクトは以下のスコアで評価する（各5点満点）：

| 評価軸 | 説明 |
|--------|------|
| 自動化可能性 | どこまでコードで自動化できるか |
| 初期コスト | 始めるのにいくらかかるか（低いほど高評価） |
| 収益化速度 | 収益が出るまでの期間（短いほど高評価） |
| 市場性 | 需要の大きさ |
| 既存スキルとの親和性 | 今のスキルセットで戦えるか |
